# 课件一键交付包

> 输入一个课件 PDF，一键生成完整交付物料包。

## 这是什么？

做课件电商的朋友经常面临一个问题：课件做完了，但**卖相不够**。

你需要小红书封面图吸引流量，需要逐字稿让买家知道怎么讲，需要教学设计证明专业性，还需要样机图展示「这堂课在教室里是什么样」——这些本该专注在课程内容上的时间，全花在了排版、截图、拼图这些重复劳动里。

**这个工具就是为此而生。** 给它一个 PDF，它帮你把上面这些全自动搞定。

## 你能得到什么

| 产物 | 说明 |
|------|------|
| 🖼️ **拼图封面**（3 张） | 二图版 / 五图版 / 十二图版竖版拼图，1200×1600，直接发小红书 |
| 📝 **逐字稿**（Word） | 完整的课堂讲稿，可直接用于试讲或交付买家 |
| 📐 **教学设计**（Word） | 教学流程、目标、重难点完整方案 |
| 📺 **样机展示图**（36 张） | 6 种教室场景 × 前 6 页课件，买家一眼看到上课效果 |

所有产物输出到课件 PDF 同级目录，即拿即用。

## 快速开始

### 1. 环境准备

```bash
pip install pymupdf pillow python-docx
```

### 2. 配置

```bash
cp agents/config.example.json agents/config.local.json
cp agents/stage2_content.example.json agents/stage2_content.local.json
```

然后编辑 `config.local.json`，填入：
- 课件 PDF 所在目录
- 字体文件路径（用于拼图封面中文渲染）
- 样机模板路径

### 3. 运行

```bash
# 阶段 1：生成拼图封面
python agents/stage1_collage.py

# 阶段 2：生成逐字稿 + 教学设计
python agents/stage2_docs.py

# 阶段 3：生成样机展示图
python agents/stage3_mockup.py
```

三个阶段串行执行，共享同一份 PDF 渲染结果，不会重复渲染。

### 4. 输入形式

除了 PDF，也支持：
- 已导出的 PPT 页面图
- `images/page-01.png` ~ `page-xx.png` 格式的图片目录
- 可选的参考拼图图样（影响封面风格）

## 常见用途

- 📦 课件产品交付前的**物料打包**
- 🛒 小红书/淘宝课件店铺的**商品图制作**
- 🏫 公开课/师训资源包的**配套资料生成**
- 🤖 配合 Claude Code Skill 运行时**全自动出包**

## 配置文件说明

| 文件 | 用途 |
|------|------|
| `config.local.json` | 本地路径配置（不上传 GitHub） |
| `stage2_content.local.json` | 逐字稿/教学设计的主题和风格偏好 |
| `config.example.json` | 配置模板，可参考字段说明 |
| `openai.yaml` | LLM API 配置 |

## 注意事项

- 样机模板图片（`references/样机图/`）请确认拥有商业使用或再分发权限
- 不要将真实课件路径、客户信息、API 密钥提交到公开仓库
- 如需保留含真实项目数据的版本，请在私有仓库中使用原始 Skill，本仓库仅作公开参考
